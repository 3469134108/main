# wxpython制作的pip安装程序

用的wxpython库，需要先安装：（需要有python解释器）

1.国内用户：（清华源）

```
pip install wxpython -i https://pypi.tuna.tsinghua.edu.cn/simple
```

2.直接双击打开（需要有python解释器，如果没有，点击进入官网：[python.org](python.org)）
